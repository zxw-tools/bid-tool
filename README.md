# bid-tool

这是一个用于 **工程投标报价评标分析** 的 Python 工具，支持：

- 📊 从 Excel 导入投标单位、报价、暂列金、信用分
- 🧮 按评标办法计算评标基准价（一次 / 二次 / 三次平均）
- 🎯 计算投标报价得分（支持 E1 / E2 参数）
- 📈 枚举组合，统计各投标单位中标概率
- 🏆 报价得分相同情况下，按「信用分优先，其次报价低者优先」确定中标单位

## 使用说明

1. 准备 Excel 数据文件（包含单位名称、报价、暂列金、信用分）
2. 运行 `bid_tool2_final.py`
3. 根据提示选择功能并输入参数
4. 查看控制台输出结果

## 适用场景

- 工程项目投标报价分析
- 评标规则模拟
- 报价策略研究
- 中标概率分析

---

> 本项目用于学习与辅助分析，具体投标结果以官方评标为准。
